About

Hello, this is a small website I created after discovering Neocities. Eventually it will contain random stuff that I find interesting. I currently run another website that is way messier than this one and that contains more stuff about me personally.

I'm not completely sure what I would like to put here yet, but we will see what seems interesting and go with it! I might end up just using it to share cool things created by other people.

This page is being hosted on Neocities since it's completely free and has an interesting way for me to find other people's websites.

Inspiration

Now, I'm no graphic designer nor do I claim to know what I'm doing at all when it comes to web or graphic design. However, I do enjoy making cute little websites and I would like to share some of my inspirations for this one.

The look of this site was heavily inspired by the wonderful Return of the Obra Dinn, by Lucas Pope, which I have been fascinated with the look of since playing it a while ago.

Most images are dithered thanks to an article I read about Why Your Website Should Use Dithered Images, and it just so happens to match the 1-bit aesthetic of the game as well.

Obra Dinn has many retro inspired monochrome color schemes but I stole this one because it's very comfortable on my eyes. This color scheme is titled "IBM 8503" in the settings, and the hex codes are #2e3037 and #ebe5ce.

On my links page you can also find a bunch of websites that have inspired me to create my own. Some are from Neocities and some are ones that I found elsewhere. Either way they should all be great and worth a visit!

Tools

Everything is hand written using Visual Studio Code and hosted on this great website called Neocities! Any images I use are either by me or found on the internet, original authors will always be linked. Any dithered images are created using this tool called Dither Me This.
